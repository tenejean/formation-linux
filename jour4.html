1. Structure du système de fichiers

  • / : racine du système
  • /home : répertoires des utilisateurs
  • /etc : fichiers de configuration
  • /var : logs, mails, impressions
  • /tmp : fichiers temporaires
  • /opt : logiciels additionnels
  • /usr : utilitaires et logiciels installés

2. Commandes de navigation & manipulation

CommandeDescription
cdChanger de répertoire
ls -lLister le contenu avec détails
cp source destCopier un fichier ou dossier
mv source destDéplacer ou renommer
rm -r dossierSupprimer récursivement
mkdir -p dossierCréer dossier(s)

3. Exercices pratiques

  1. Créer un dossier formation dans ton /home
  2. Créer deux sous-dossiers scripts et notes
  3. Copier /etc/passwd dans notes
  4. Déplacer le fichier passwd vers formation
  5. Supprimer le dossier notes

4. Mini défi

Crée une arborescence comme suit, avec une seule commande :


/home/utilisateur/projets/linux/{scripts,configs,tests}

Puis copie un fichier de ton choix dans chaque dossier et vérifie avec tree.

Résumé synthétique

  • cd, ls, pwd : navigation
  • cp, mv, rm : manipulation fichiers
  • mkdir, rmdir : création/suppression répertoires
  • tree : afficher l’arborescence
